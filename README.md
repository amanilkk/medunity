# 🏥 Système de Gestion de Clinique Privée

## 📌 Description

Ce projet est une application web permettant la gestion complète d’une clinique privée.
Il centralise les activités médicales, administratives, financières et logistiques dans une seule plateforme afin d’améliorer l’organisation interne et la qualité des services.

---

## 🎯 Objectifs

* Digitaliser les opérations de la clinique
* Réduire les erreurs humaines
* Améliorer la gestion des patients
* Faciliter la communication entre les services
* Assurer la sécurité des données médicales

---

## ⚙️ Fonctionnalités principales

### 👨‍⚕️ Médecin

* Consultation des dossiers patients
* Ajout de diagnostics
* Prescription d’ordonnances
* Suivi des analyses

### 🧑‍💼 Secrétaire

* Gestion des rendez-vous
* Enregistrement des patients
* Gestion des lits

### 💊 Pharmacien

* Gestion du stock
* Traitement des ordonnances

### 🧪 Laboratoire

* Gestion des analyses
* Envoi des résultats

### 💰 Comptable

* Facturation
* Suivi des paiements
* Rapports financiers

### ⚙️ Administrateur

* Gestion des utilisateurs
* Gestion des rôles
* Sécurité et audit

### 🏢 Gestionnaire des moyens généraux

* Gestion des lits et chambres
* Gestion des équipements
* Gestion des stocks
* Gestion des maintenances
* Gestion des fournisseurs et commandes

### 👥 Ressources Humaines (RH)

* Gestion des employés
* Gestion des présences et absences
* Gestion des congés
* Gestion des plannings
* Gestion des contrats
* Calcul des salaires

---

## 🛠️ Technologies utilisées

* Frontend : HTML, CSS, JavaScript
* Backend : PHP
* Base de données : MySQL
* Serveur local : XAMPP
* Versioning : GitHub

---

## 🔐 Sécurité

* Authentification des utilisateurs
* Gestion des rôles (RBAC)
* Journalisation des actions (logs)
* Protection des données médicales

---

## 🚀 Installation

1. Cloner le projet :

```bash
git clone https://github.com/amanilkk/clinic-management-system.git
```

2. Placer le projet dans le dossier `htdocs` (XAMPP)

3. Importer la base de données MySQL

4. Lancer Apache et MySQL

5. Accéder via :

```
http://localhost/nom-du-projet
```
